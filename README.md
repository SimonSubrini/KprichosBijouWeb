# Kprichos Bijou 🛍️✨

Bienvenido al repositorio oficial de **Kprichos Bijou**, una tienda online dedicada a la venta de accesorios únicos de resina epóxica, llaveros personalizados y souvenirs creados de forma artesanal.

## 🚀 Tecnologías Utilizadas

- **Frontend:** React, Vite, TailwindCSS
- **Backend & CMS:** Sanity.io (Headless CMS)
- **Gestión de Estado:** Zustand
- **Despliegue:** Vercel

## 🌟 Características Principales

- **Catálogo Dinámico:** Integración en tiempo real con Sanity para gestionar productos en stock y a pedido.
- **Personalización de Productos:** Opciones configurables desde el CMS que ajustan dinámicamente el precio en el frontend.
- **Checkout vía WhatsApp:** Sistema de carrito de compras que deduce el stock en la base de datos y redirige al cliente a WhatsApp con un resumen detallado de su pedido.
- **Auto-despliegue:** Configurado mediante GitHub Actions / Vercel para reflejar los cambios instantáneamente.

## 🛠️ Instalación Local

Si deseas correr este proyecto en tu entorno local:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/SimonSubrini/KprichosBijouWeb.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```
